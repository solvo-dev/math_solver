# math_solver

Ein einfacher, clientseitiger Mathe-Chatbot auf Deutsch. Unterstützt:

- Additionen (z. B. `3 + 4`, `5 plus 8`)
- Subtraktionen (z. B. `10 - 7`, `12 minus 5`, `subtrahiere 2 von 9`)
- Multiplikationen (z. B. `6 * 5`, `3 × 4`, `3x4`, `5 mal 8`, `multipliziere 7 mit 6`)
- Divisionen (z. B. `8 / 2`, `9 ÷ 3`, `10 : 2`, `12 geteilt durch 4`, `dividiere 15 durch 3`)

Öffne einfach `index.html` im Browser und stelle deine Rechenfrage im Chat.
